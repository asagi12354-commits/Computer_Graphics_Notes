本文严格基于NeRF原版论文《NeRF: Representing Scenes as Neural Radiance Fields for View Synthesis》内容，厘清NeRF、光线追踪、光线步进三者的核心关联，完整推导论文核心公式，深入解析NeRF渲染管线的底层逻辑，助力精准理解这一开创性技术的核心原理。

# 一、核心概念厘清：三者的关系并非对立，而是「框架\-方法\-表示」的深度绑定

很多人易误解NeRF替代了光线追踪或光线步进，实则不然：**NeRF的渲染核心完全继承光线追踪的物理框架，以光线步进作为数值求解方法，最终用神经网络完成场景表示的范式革新**。三者的本质、边界及关联如下：

## 1\.1 光线追踪（Ray Tracing）：顶层物理渲染框架

光线追踪是计算机图形学的核心渲染范式，本质是模拟真实世界光线的传播规律，以「光线」为最小单位计算像素颜色。其核心逻辑采用「逆向光线追踪」（从相机/人眼出发，而非光源）——因只有进入相机的光线才对成像有贡献：对图像的每个像素，发射一条穿过该像素的光线，追踪这条光线在场景中的传播，计算其最终携带的颜色，作为该像素的取值。

光线追踪主要分为两大分支：

- 表面光线追踪：传统3D渲染的主流方式，场景用三角形网格表示，核心是「光线与三角形的解析求交」，找到交点后计算BRDF光照，一条光线通常只需1个有效交点。

- 体光线追踪（体渲染）：针对烟雾、云、半透明介质等无显式表面的场景，核心是「计算光线穿过介质时，所有采样点的发光与衰减的累积」。NeRF完全继承这一分支的物理框架，论文中也明确引用了体渲染的奠基工作（Kajiya 1984、Max 1995）。

## 1\.2 光线步进（Ray Marching）：光线追踪的具体数值实现方法

光线步进是体光线追踪/隐式场景渲染的核心求解算法，专门解决「无显式表面的场景，无法用解析法求光线与表面交点」的问题。其核心逻辑是：沿着发射的光线，从近平面到远平面，按一定规则一步步“步进”采样离散的3D点，计算每个采样点对最终像素颜色的贡献，最终通过积分/求和合成完整的像素颜色。

它与光线追踪的关系：光线步进是实现光线追踪的工具，而非平行技术。类比来说，“光线追踪”是“从A到B”的目标，“光线步进”就是“开车”这种实现目标的具体方式。

## 1\.3 NeRF：用神经网络参数化场景，复用光线追踪\+光线步进的渲染管线

NeRF的核心创新并非发明新的渲染框架，而是革新了场景的表示方式，对比传统体渲染与NeRF的场景表示：

- 传统体渲染：场景用离散的3D体素网格存储，每个体素记录固定的密度和颜色，存储成本随分辨率三次方增长，无法表示超高精度场景。

- NeRF：场景用一个全连接神经网络（MLP）表示，输入连续的5D坐标，输出该位置的密度和视角相关颜色，是一种无限分辨率、连续、存储成本极低的隐式场景表示。

NeRF的渲染管线100%遵循体光线追踪的物理框架，用光线步进完成数值积分，最终实现新视角的渲染。

# 二、三者的协作关系：NeRF渲染的完整流程

结合NeRF论文核心逻辑，用一个完整的像素渲染流程，串联三者的协作过程，清晰呈现NeRF的工作机制：

1. 光线追踪启动：对目标图像的每个像素，从相机光心发射一条穿过该像素的光线，定义光线方程 $r(t) = o + t*d$（$o$是相机原点，$d$是光线单位方向向量，$t$是沿光线的深度），确定光线的近/远边界 $[t_n, t_f]$。

2. 光线步进采样：沿着这条光线，通过分层采样策略，步进采样一系列3D点 $r(t_i)$，这一步是光线步进的核心操作。

3. NeRF场景查询：把每个采样点的3D坐标和光线方向，输入训练好的MLP，查询得到该点的体密度$\sigma_i$和RGB颜色$c_i$。

4. 体光线追踪积分：用体渲染的Alpha合成公式，把所有采样点的贡献累积起来，计算出这条光线最终的颜色，也就是对应像素的取值。

5. 训练优化：因整个渲染流程完全可微，可通过「渲染颜色与真实图像像素的误差」反向传播，优化MLP的权重，让它学会正确的场景表示。

# 三、NeRF论文核心原理与公式全推导

以下推导严格对应NeRF原版论文内容，从核心定义到最终离散公式，完整解析每一步的物理意义和数学逻辑，贴合论文原文推导思路。

## 3\.1 核心定义：5D神经辐射场（论文第3节）

NeRF将一个静态场景，定义为一个连续的5D向量值函数：

$F_{\Theta}: (x, d) \to (c, \sigma)$

其中各参数的具体含义的如下：

### 输入参数

- $x=(x,y,z)$：空间中任意一点的3D坐标；

- $d=(\theta,\phi)$：观察该点的2D视角方向，论文中实际用3D单位笛卡尔向量表示，取值范围在$[-1,1]$之间。

### 输出参数

- $\sigma$：该点的体密度，标量，且非负。物理意义是单位长度内光线被吸收的概率，是场景的固有属性，仅和空间位置$x$有关，与视角无关（一个点的不透明度不会因观察方向改变而变化）。

- $c=(r,g,b)$：该点在视角$d$下的发射辐射亮度，RGB三通道，取值范围$[0,1]$。物理意义是该点向视角$d$方向发出的光线颜色，同时和位置$x$、视角$d$有关，用于表示非朗伯体的视角相关效果（如高光、反射）。

### 网络结构设计（对应论文图7）

论文中对MLP的结构做了关键约束，以保证多视角一致性，具体结构如下：

1. 先对位置编码$\gamma(x)$经过8层全连接层（ReLU激活，256通道），输出体密度$\sigma$和256维特征向量；

2. 将256维特征向量与视角编码$\gamma(d)$拼接，经过1层128通道的全连接层（ReLU激活），最终输出视角相关的RGB颜色。

## 3\.2 核心渲染公式：体渲染积分的完整推导（论文第4节）

这部分是NeRF与光线追踪、光线步进绑定的核心，论文中的公式\(1\)源自经典体渲染的辐射传输方程，以下从头推导，明确每一步的物理意义和数学逻辑。

### 步骤1：光线的定义

我们要渲染的相机光线，参数化为：

$r(t) = o + t d$

各参数说明：

- $o$：相机光心（光线起点）；

- $d$：光线的单位方向向量；

- $t$：沿光线的深度参数，$t \in [t_n, t_f]$，$t_n$是近平面，$t_f$是远平面，即光线在场景中的有效传播范围。

### 步骤2：辐射传输微分方程

光线在参与介质中传播时，辐射亮度的变化由两部分组成，结合物理规律可得微分方程：

1. 吸收衰减：光线穿过介质时，被介质吸收，亮度随传播距离下降，微分形式为：$dL(t) = -\sigma(t) L(t) dt$，其中$\sigma(t)$是体密度，代表单位长度的吸收概率，$\sigma$越大，光线被吸收得越快。

2. 自发光贡献：介质本身会向光线方向发射光线，为亮度带来增量，微分形式为：$dL(t) = \sigma(t) c(t) dt$，其中$c(t)$是该位置介质向光线方向发射的颜色，即NeRF中MLP输出的RGB值。

将两者合并，得到沿光线的辐射传输一阶线性非齐次常微分方程：

$\frac{dL(t)}{dt} = -\sigma(t) L(t) + \sigma(t) c(t)$

### 步骤3：求解微分方程，得到连续积分公式（论文公式1）

采用积分因子法求解该ODE，目标是得到相机在$t=t_n$处接收到的总亮度$C(r)$（即像素颜色），具体步骤如下：

1. 整理方程，将其转化为标准线性ODE形式：$\frac{dL(t)}{dt} + \sigma(t) L(t) = \sigma(t) c(t)$；

2. 构造积分因子：$\mu(t) = \exp\left( \int_{t_n}^t \sigma(s) ds \right)$，方程两边同时乘以积分因子，左边可转化为乘积的导数：$\frac{d}{dt} \left( \mu(t) L(t) \right) = \mu(t) \sigma(t) c(t)$；

3. 两边从$t_n$到$t_f$积分，代入边界条件：光线在$t=t_f$（远平面之外）没有任何介质，因此$L(t_f)=0$（无来自无穷远的光），积分结果为：$0 - \mu(t_n) L(t_n) = \int_{t_n}^{t_f} \mu(s) \sigma(s) c(s) ds$；

4. 化简计算：其中$\mu(t_n) = \exp\left( \int_{t_n}^{t_n} \sigma(s) ds \right) = 1$，因此可得：$L(t_n) = \int_{t_n}^{t_f} \exp\left( -\int_{t_n}^s \sigma(\tau) d\tau \right) \sigma(s) c(s) ds$。

对应论文的标准形式（公式1）：论文中定义累积透射率$T(t)$——光线从$t_n$传播到$t$，没有被任何介质吸收的概率：

$T(t) = \exp\left( -\int_{t_n}^t \sigma(r(s)) ds \right)$

最终得到论文中的体渲染积分公式（核心渲染公式）：

$C(r) = \int_{t_n}^{t_f} T(t) \sigma(r(t)) c(r(t), d) dt$

该公式是NeRF渲染的核心，也是体光线追踪的物理基础。

### 步骤4：光线步进的离散化实现（论文公式2、3）

上述积分是连续的，无法直接在计算机中计算，因此需要用光线步进的方式做数值积分，这也是光线步进在NeRF中的核心作用，具体实现如下：

#### 1\. 分层采样（论文公式2）

论文未采用固定间隔的均匀采样，而是采用分层随机采样，保证MLP能在连续位置上被查询，适配连续场景表示：将$[t_n, t_f]$划分为$N$个等宽的区间，在每个区间内均匀随机采样一个点，采样公式为：

$t_i \sim \mathcal{U}\left[ t_n + \frac{i-1}{N}(t_f - t_n), t_n + \frac{i}{N}(t_f - t_n) \right]$

这一步是光线步进的核心操作：沿着光线，一步步划分区间、采样3D点。

#### 2\. 离散积分公式（论文公式3）

有了离散的采样点，论文采用Max 1995提出的体渲染求积规则，将连续积分转化为离散求和，具体推导如下：

定义相邻采样点的间距$\delta_i = t_{i+1} - t_i$，假设在区间$[t_i, t_{i+1}]$内体密度$\sigma_i$为常数，则：

- 该区间的不透明度（Alpha值）：$\alpha_i = 1 - \exp(-\sigma_i \delta_i)$，代表光线在该区间被吸收的概率；

- 累积透射率：$T_i = \exp\left( -\sum_{j=1}^{i-1} \sigma_j \delta_j \right)$，代表光线从起点到达$t_i$处，还未被吸收的概率。

最终，离散的像素颜色合成公式为（对应论文公式3）：

$\hat{C}(r) = \sum_{i=1}^N T_i \left( 1 - \exp\left( -\sigma_i \delta_i \right) \right) c_i$

该公式本质是计算机图形学的Alpha合成（Over算子），从前往后依次合成每个采样点的颜色贡献，完成光线步进的数值积分。

## 3\.3 论文的两大核心优化技巧

基础的NeRF存在两个关键问题：MLP难以拟合高频细节、均匀采样效率极低。论文提出两个核心优化技巧，彻底解决了这两个问题，保障了NeRF的渲染质量和效率。

### 3\.3\.1 位置编码（Positional Encoding，论文5\.1节）

论文发现，直接输入原始的$xyz\theta\phi$坐标，MLP会严重偏向学习低频信息，无法还原场景的精细几何和纹理（即神经网络的频谱偏置）。因此引入位置编码，将低维坐标映射到高维的正弦余弦空间，让MLP更容易学习高频函数。

论文的位置编码公式（公式4）：

$\gamma(p) = \left( \sin\left(2^0 \pi p\right), \cos\left(2^0 \pi p\right), \cdots, \sin\left(2^{L-1} \pi p\right), \cos\left(2^{L-1} \pi p\right) \right)$

参数说明及核心作用：

- 输入$p$是标量（如$x$坐标的一个分量），输出是$2L$维的高维向量；

- 对3D位置$x$的每个分量，取$L=10$，覆盖从$\pi$到$2^9\pi$的频率；

- 对3D视角方向$d$的每个分量，取$L=4$，因为视角相关的颜色变化频率更低；

- 核心作用：两个距离极近的点，原始坐标差异极小，但经过高频编码后，高维特征差异极大，让MLP能轻松区分并学习到精细的细节。

### 3\.3\.2 分层体积采样（Hierarchical Volume Sampling，论文5\.2节）

均匀采样的效率极低：大部分采样点落在空旷空间或被遮挡区域，对最终像素颜色几乎没有贡献。论文提出粗网络\+细网络的分层采样策略，让采样点集中在对渲染有贡献的区域，具体流程如下：

1. 粗网络（Coarse Network）：先在光线上均匀采样$N_c$个点，输入粗网络得到每个点的$\sigma_i$和$c_i$，计算每个点的权重：$w_i = T_i \left( 1 - \exp\left( -\sigma_i \delta_i \right) \right)$，其中$w_i$是该点对最终颜色的贡献大小，权重越大，越可能是物体表面。

2. 生成概率分布：将权重归一化，得到沿光线的概率密度函数（PDF）：$\hat{w}_i = \frac{w_i}{\sum_{j=1}^{N_c} w_j}$。

3. 重要性采样：用逆变换采样，从这个PDF中再采样$N_f$个点，这些点会集中在权重高的区域（物体表面附近）。

4. 细网络（Fine Network）：将粗采样的$N_c$个点和细采样的$N_f$个点合并，输入细网络，查询所有点的$\sigma$和$c$，再用公式3计算最终的像素颜色$\hat{C}_f(r)$。

### 3\.3\.3 损失函数（论文公式5）

为了同时优化粗网络和细网络，论文采用了双边均方误差损失，公式如下：

$\mathcal{L} = \sum_{r \in \mathcal{R}} \left[ \left\| \hat{C}_c(r) - C(r) \right\|_2^2 + \left\| \hat{C}_f(r) - C(r) \right\|_2^2 \right]$

参数说明：

- $\mathcal{R}$：一个batch内的光线集合；

- $C(r)$：真实图像的像素颜色；

- $\hat{C}_c(r)$：粗网络渲染的颜色，$\hat{C}_f(r)$：细网络渲染的颜色。

同时优化粗网络，是为了让粗网络输出的权重分布更准确，从而生成更合理的采样PDF，把采样点分配到正确的区域。
